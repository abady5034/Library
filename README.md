Library Management System :
A simple desktop application built using Java & JavaFX for managing a small library system.
The application allows users to store and manage book names using a .txt file.

⸻

 Features
	•	 Add new books
	•	 View all stored books
	•	 Delete books
	•	 Data persistence using .txt file

⸻

 Technologies Used
	•	Java 
	•	don't use oop (basics only)
	•	JavaFX
	•	File Handling (.txt)

⸻

 How to Run
	1.	Make sure you have Java (JDK 8 or higher) installed
	2.	Download JavaFX SDK (if not included)
	3.	Open the project in your IDE (IntelliJ / Eclipse)
	4.	Run the Main.java file

⸻

 Notes
	•	All book names are stored locally inside a .txt file
	•	Make sure the file exists before running the program
	•	You can edit or clear the file manually if needed

⸻

 Future Improvements
	•	Search functionality
	•	Save additional details (author, category, etc.)
	•	Use database instead of .txt file
